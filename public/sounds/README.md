# Sons de Notificação

Este diretório contém arquivos de som para notificações do chat.

## notification.mp3

Som curto de notificação para novas mensagens.

### Como substituir o som:

1. Adicione um arquivo `notification.mp3` nesta pasta
2. O som deve ser curto (menos de 1 segundo)
3. Volume moderado
4. Formato: MP3

### Onde conseguir sons gratuitos:

- [Freesound.org](https://freesound.org)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [Zapsplat.com](https://www.zapsplat.com)

### Configuração atual:

Se o arquivo `notification.mp3` não existir, o sistema usará um som gerado 
automaticamente pela Web Audio API (um tom suave de 800Hz por 0.3s).
